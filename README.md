## サービス概要

https://alc-app.matsu.works/

## 機能一覧
* ユーザー情報の登録 / ログイン機能
* Googleアカウントでのユーザー登録 / ログイン機能
* リストのCRUD機能
* リストの検索機能
* 画像のアップロード / 削除機能
* 画像からリスト情報の取得・表示(リレーション)
* ページネーション機能
* レスポンシブ対応
